Louarch
=======

Rewrite of bash script with golang for better debugging and single binary.

Requirements
------------

-	Go 1.23 or higher
-	Wofi 1.5.1 or higher
-	wallust 3.4.0 or higher

Installation
------------

-	create binary using `make build`
-	place binary in $PATH
-	run `louarch --help` to see usage
